  • Resolving Issues

    If you happen to experience trouble follow steps below.

    1. Power the device off and back on.
    2. Manually select network : ‘Carrier’ will appear in the setting menu > Tap on carrier and toggle Automatic to OFF to allow for manual selection of GSM carrier > select on available carrier.
    3. If the issue is not resolved contact Worldwide Care.
      Chat : www.sprint.com/swwsupport
      Call : +1-817-698-4199 (when prompted select option 3 to report trouble while outside of U.S. )
      • City, country you are traveling in
      • Contact number (hotel, etc.)
      • Error messages displayed on device or heard while calling.
